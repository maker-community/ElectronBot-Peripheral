 1. 侧面天线
 2. 右手武器
 3. 左手盾牌
 4. 高达翼装
 5. 恶魔翅膀
 6. 头顶鹿角
 7. 胸口手势孔装饰--2023.1.14
 8. 笔套--2023.1.15
 9. 兔子装饰（发箍、尾巴）--2023.1.30
   
### 安装方式：
  1.  #### 角、翅膀使用 胶水、玻璃胶固定
  2.  #### 手部兼容原版，直接安装即可
  3.  #### 耳机独立配件，可以拆卸
  4.  #### 笔套独立配件，安装在后背安装孔上
