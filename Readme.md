## ElectronBot的外设

### 主体使有稚辉君的 ElectronBot 模型。
 1. 增加侧面耳朵
 2. 右手改武器
 3. 左手盾牌
 4. 背后高达翼装
 5. 头顶鹿角
 6. 胸口手势孔装饰
   
### 安装方式：
  1.  #### 角、翅膀使用 胶水、玻璃胶固定
  2.  #### 手部兼容原版，直接安装即可
  3.  #### 耳机独立配件，可以拆卸
